# HTML & CSS

## 引言

从这次任务开始，我们正式进入前端的学习。

**HTML**是一种标记语言，使用成对的标记标签来描述网页，比如`<a></a>`，是网页的骨架

**CSS**是描述应该如何显示HTML元素，是网页的视觉呈现

使用HTML+CSS就能构造一个基础的网页，而现在，利用HTML5和CSS3就能写出非常漂亮和强可交互性的网页

## 任务目标

- 基础：掌握常用的 HTML 标签
- 基础：掌握常用的 CSS 属性
- 附加：掌握`flex`布局
- 附加：了解HTML5和CSS3
